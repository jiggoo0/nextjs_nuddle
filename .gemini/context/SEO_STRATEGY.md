# 🚀 SEO DOMINANCE STRATEGY: ช.สหชัย เกี๊ยวปูหมูแดง จ.ตาก

## 🎯 Primary Goal

ยึดพื้นที่หน้าแรก Google ในจังหวัดตาก สำหรับทุกคำค้นหาที่เกี่ยวข้องกับ "บะหมี่", "ร้านอาหาร", และ "ของดีเมืองตาก"

## 🔑 Target Keywords (Local SEO)

- **High Intent:** `บะหมี่เกี๊ยวปู ตาก`, `ร้านบะหมี่อร่อย ตาก`, `ช.สหชัย เกี๊ยวปูหมูแดง`, `ของกินเมืองตาก`
- **Location-Based:** `ร้านอาหารใกล้สะพานกิตติขจร`, `ร้านอร่อยจังหวัดตาก`, `บะหมี่เกี๊ยวปู สะพานกิตติขจร`
- **Dish-Specific:** `หมูแดงย่างเตาถ่าน ตาก`, `เกี๊ยวปูเน้นๆ ตาก`, `บะหมี่ทำเอง เมืองตาก`

## 🌟 Rich Snippets Plan (Schema.org)

ต้องปรากฏข้อมูลเหล่านี้ในหน้าผลการค้นหา:

1. **Stars Rating:** แสดงคะแนนรีวิว (เช่น 4.9/5 จาก 520 รีวิว)
2. **Price Range:** ช่วงราคา (฿ - ฿฿)
3. **Opening Hours:** สถานะเปิด-ปิดปัจจุบัน
4. **LocalBusiness Map:** เชื่อมต่อกับ Google Maps (ลิงก์: https://maps.app.goo.gl/KLdnUCffDN6RZpTT8)

## 🛠️ Technical SEO Checklist (Next.js 16)

- [ ] **Metadata API:** ตั้งค่า Dynamic Metadata สำหรับทุกหน้า
- [ ] **JSON-LD Implementation:** ใส่ Structured Data ใน `layout.tsx`
- [ ] **Open Graph (OG):** ภาพพรีวิวเวลาแชร์ใน Facebook (เน้นรูปบะหมี่น่าทาน)
- [ ] **Image Alt Tags:** ใส่คีย์เวิร์ดในรูปภาพทุกรูป (เช่น `alt="บะหมี่เกี๊ยวปู ช.สหชัย จ.ตาก"`)
- [ ] **Sitemap & Robots.txt:** สร้างอัตโนมัติเพื่อการ Index ที่รวดเร็ว
